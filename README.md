## Lottery Picker

### Description

Input the countdown timer and start , after coutdown to 0 will randomly select a winner 
** the generater will auto generate list , default set is 9 members **

### Installation

step 01:
git clone https://github.com/ericfei0521/lottery.git

step 02:
cd lottery

step 03:
npm install

### activate

npm run start

### Using the app
Local: 
http://localhost:3000
